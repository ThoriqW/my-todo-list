
# My-Todo-List

This app using MongoDB to store the data


## Authors

- [@ThoriqW](https://www.github.com/ThoriqW)


## Run Locally

Clone the project

```bash
  git clone https://github.com/ThoriqW/my-todo-list
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  node app.js
```
